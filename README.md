
# Seedling 🌱

A personal project to track plant sowing, transplanting and harvesting.
- PWA
- Developed for mobile and desktop

## Tech Stack
- Frontend: Vue, TypeScript
- Backend: Deno (TypeScript), Hono
- Database: PostgreSQL (Neon)
- Auth: OAuth2 / JWT
- Deployment: Fly.io
